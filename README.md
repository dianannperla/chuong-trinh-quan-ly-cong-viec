# Chương Trình Quản Lý Công Việc

Chào mừng bạn đến với Chương Trình Quản Lý Công Việc - một framework frontend chuyên dụng cho việc xây dựng các ứng dụng quản lý công việc một cách nhanh chóng và linh hoạt.

## Tính năng nổi bật
- **Giao diện người dùng thân thiện**: Thiết kế trực quan giúp người dùng dễ dàng sử dụng.
- **Quản lý nhiệm vụ**: Tạo, sửa, và xóa nhiệm vụ một cách dễ dàng.
- **Phân loại nhiệm vụ**: Người dùng có thể phân loại nhiệm vụ theo nhiều tiêu chí khác nhau như độ ưu tiên, hạn chót, và trạng thái.
- **Tích hợp lịch**: Nhắc nhở và hiển thị nhiệm vụ sắp tới.

## Cài đặt
1. Clone repo:
   ```bash
   git clone https://github.com/yourusername/chuong-trinh-quan-ly-cong-viec.git
   ```
2. Cài đặt các gói cần thiết:
   ```bash
   npm install
   ```
3. Chạy ứng dụng:
   ```bash
   npm start
   ```
